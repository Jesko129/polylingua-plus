# PolyLingua+ 🌐

## AI-Powered Mobile Translator with OCR and Speech
A comprehensive mobile translation application supporting Mandarin, English, and Filipino with advanced features including OCR, speech recognition, and text-to-speech.

### Features
✅ **Text-to-Text Translation** - Translate words, phrases, and sentences
✅ **Speech-to-Text Translation** - Speak and translate in real-time
✅ **Text-to-Speech Output** - Hear translations pronounced aloud
✅ **OCR Image Translation** - Extract and translate text from images
✅ **Language Auto-Detect** - Automatically identify input language
✅ **Bidirectional Translation** - All language combinations supported

### Supported Languages
- 🇨🇳 Mandarin Chinese
- 🇬🇧 English
- 🇵🇭 Filipino

### Tech Stack
- **Frontend**: Flutter (Dart)
- **Backend**: Node.js with Express
- **OCR**: ML Kit Vision / google_mlkit_text_recognition
- **Speech**: speech_to_text & flutter_tts
- **State Management**: Provider
- **Translation**: Google Translate API
