# PolyLingua+ Flutter Design System

## Color Scheme
```
Primary Dark:     #0f3460
Secondary Dark:   #16213e
Accent Cyan:      #00d4ff
Text White:       #ffffff
Text Gray:        #888888
Error Red:        #ff6b6b
```

## UI Screens

### 1. Translator Screen
- Language selector (From/To)
- Text input with microphone button
- Swap languages button
- Translation output
- Translate action button
- Text-to-speech button

### 2. OCR Screen
- Image preview area
- Camera button
- Gallery button
- Extract text button
- Extracted text display
- Language selector for translation
- Translate button
- Translated text display

### 3. History Screen
- List of translation records
- Delete individual record button
- Clear all history button

### 4. Settings Screen
- Auto-Detect Language toggle
- Text-to-Speech toggle
- Dark Mode toggle
- About section

## Typography
- App Bar: Bold, 18pt
- Section Headers: Bold, 16pt, Cyan color
- Body Text: Regular, 14pt, White
- Subtitle: Regular, 12pt, Gray
